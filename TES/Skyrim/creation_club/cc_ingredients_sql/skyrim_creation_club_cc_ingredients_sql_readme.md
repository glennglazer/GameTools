# Purpose and Action

This directory is intentionally empty.  No CC alchemy ingredients SQL loader
is needed.

## Why

All CC alchemy ingredients (Healing Salts, Ironwood Fruit, Corkbulb Root,
Angler Larvae) are already present in `skyrim_alchemy_ingredients` from the
main Skyrim alchemy scrape.  The food items that appear under "Ingredients"
headings on CC content pages but are not usable in alchemy are excluded from
the database entirely.

See `cc_ingredients_json/` for the full rationale.
